## **Revenge of the PETSCII Robots**

- Added a function that prevents deployable items being placed on wall tiles
  - Items are pushed back to the tile they were dropped from.
- Added a reusable function for timers so that they don't have to be created from scratch each time.
- Added new timer functions to deployable items.
- Improved goo bomb code by reducing complexity.
- Redid and improved hoverbot movement.
- Rewrote and greatly improved movement functions for all instances.
- Rewrote and greatly improved object pushing functions.
- Fixed an old error of incorrectly sized hitboxes that prevented proper movement collisions.
- Added draw hitbox function.
- Cleaned up inheritance and unused variables.

## **WhiteHex Library**

- Expanded the test world map to cover a basic exploration loop in a retro JRPG.
